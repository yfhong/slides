== Ops

运维:: 保证产品能够正确无误的运行以支持整体商业目标。
....
运行:: 业务部署，配置管理

准确无误:: 系统监控，异常检测

支持商业目标:: 额外的技术保障，可用率、安全、流程规范、响应时间等等
....
